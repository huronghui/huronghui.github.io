GreenDao 初步使用

本章我们主要初步学习 GreenDao ，我们先了解一下 GreenDao 是什么

###什么是GreenDao
我相信看过android开发书籍的人一定都知道有这么一章，就是讲 SQLite，然而，在我们实际开发中，我们需要编写很多额外的东西，比如：编写SQL 语句以及解析查询结果等，这是一件很耗时耗力的过程，所以，一些适用于 Android 的 ORM 框架就应运而生了，比如： OrmLite、SugarORM、Realm 以及 本文的重点 GreenDao ，下面我就将结合代码总结一下在 Android Studio 中怎么使用 GreenDao

###让我们开始吧

###1. 配置模块
我们需要先配置我们将要存放数据库的地方，一般都是在 src/main 目录下新建一个和 java 同层级的[java-gen] 目录，存放 greenDa 生成的 Bean、DAO、DaoMaster、DaoSesson等

###2.配置依赖

在我们的 android 工程的（APP）的build.gradle 中，配置我们刚才所新建的目录，以及添加 GreenDao 的支持包

###3.新建 GREENDAO GENERATOR 模块
这一阶段都是纯 java 工程，通过 File -> New -> New Module -> Java Library -> 填写相应的包名与类名 -> Finish.


配置 daoexamplegenerator 工程的 build.gradle，添加 dependencies.

其实这阶段的工作都可以熊别的工程中直接拖过来直接使用，无需每次都重新创建一次
